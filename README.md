# Vrinda-Sales-Dashboard

An interactive Microsoft Excel dashboard developed to analyze Vrinda Store's annual sales performance for 2022. The dashboard provides insights into customer demographics, sales trends, order status, sales channels, and regional performance, enabling data-driven business decisions.

# Project Highlights

•	Built a fully interactive dashboard using Microsoft Excel.
•	Analyzed annual sales performance across multiple business dimensions.
•	Created dynamic charts, slicers, and KPIs for easy data exploration.
•	Enabled filtering by Month, Sales Channel, and Product Category.
•	Designed a clean and user-friendly dashboard for quick business insights.

# 📌 Key Insights

•	Women contributed a significantly higher share of total sales compared to men.
•	Amazon generated the highest number of orders among all sales channels.
•	Maharashtra recorded the highest sales, followed by Karnataka and Uttar Pradesh.
•	Adult customers placed the highest number of orders across both genders.
•	More than 90% of orders were successfully delivered, indicating strong order fulfillment performance.
